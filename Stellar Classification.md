# Abstract
Classification for stars
# Size
The first name sets the size of the star, although the first letter can be used instead:
- [Zora](https://en.wikipedia.org/wiki/Zora_(name)) (Z) - Small
- [Aya](https://en.wikipedia.org/wiki/Aya_(given_name)) (A) - Standard
- [Eirian](https://www.behindthename.com/name/eirian) (E) - Large
- [Säde](https://en.wiktionary.org/wiki/s%C3%A4de) (S) - Massive
- [Galadriel](https://en.wikipedia.org/wiki/Galadriel) (G) - Supermassive
- [Melchior](https://www.behindthename.com/name/melchior) (M) - Hyper massive
# Colour
Secondly is one or two letters for the colour:
- B - Blue
- BW - Blue-white
- W - White
- WY - White-yellow
- Y -Yellow
- YO - Yellow-orange
- O - Orange
- OR - Orange-red
- R - Red
# Aether
Next is the observed density of [[Aether|aether]]:
- H - High
- N - Normal
- L - Low
- U - Unknown

# Special
Additional symbols can be added for special circumstances:
- [Cansu](https://en.wikipedia.org/wiki/Cansu) (C) - Life signs in the system
- [Tuwile](https://www.babynamespedia.com/meaning/Tuwile) (T) - Black hole
